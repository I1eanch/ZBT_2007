# Commands
- `npm install` — чистая установка по lockfile.
- `npm run dev` — Astro dev server.
- `npm run build` — static build в `dist/`.
- `npm run preview` — локальный production preview.
- `npm run validate:content` — проверка структуры landing content.
- `npm run check` — `astro check`.
- `npm run test:e2e` — Playwright desktop+mobile.
- `npm run verify` — полный release gate.
- `npm run smoke:container` — smoke уже запущенного контейнера.
